In the Beginning: There Was a Sandbox

Mar 15th, 2013

Please allow myself to introduce… myself.

This page is a sandbox for me to experience with features of the Octopress platform.

Octopress is built atop Jekyll, the static site generator powering GitHub pages. It and utilizes by default the RDiscount markdown engine plus Liquid templating. RDiscount is based on standard Markdown by John Gruber. Liquid provides lots of nifty filters and tags.

Markdown is nice for all kinds of basic text decoration. For instance, you can indicate inline code with backticks.
